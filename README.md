# Project "Haru" API Documentation

---

## 개요

패스트캠퍼스 팀프로젝트 "하루한장"

하루에 한번, 사진과 함께 글을 기록할 수 있는 개인 일기 서비스.

"Haru"의 API 문서입니다.

변경사항 발생 시 계속 업데이트 됩니다.

## API URL

형식 :

`https://<domain name>/api/v1`

ex\)

`https://haru.jeeyong.kr/api/v1`
